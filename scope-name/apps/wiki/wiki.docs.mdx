This is a wiki application component, composed of indpendent components.

Everything in this app is a component - UI(React components), API(Node), Content(MDX).

Configure the app plugin on your workspace:

```bash
bit use org.scope-name/apps/wiki
```

### Run the application

You can run your application on a separate port to see it outside of the Bit workspace

```bash
bit run wiki
```
