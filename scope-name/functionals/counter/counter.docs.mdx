import { Counter } from './counter';

The counter component provides the UI and functionality to embiggen or shrink a number.

### Component usage

It's necessary to bring your own state management to use this component.
This component leverages Custom Hooks to allow you to tap into pre-defined functionality.

```js
const WrappedCounter = ({ children }) => {
  const { count, setDecrement, setIncrement } = useCounter(0);
  return (
    <Counter propCount={count}>
      <Counter.Increment
        handleIncrement={() => setIncrement(1)}
        disabled={false}
      ></Counter.Increment>
      <Counter.Decrement
        handleDecrement={() => setDecrement(1)}
        disabled={false}
      ></Counter.Decrement>
      <Counter.Count count={count}></Counter.Count>
    </Counter>
  );
};

<WrappedCounter />;
```
